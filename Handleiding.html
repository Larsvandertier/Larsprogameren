Handleiding php

Variable

  1. Als je een woord wilt schrijven met php gebruik dan de volgende code:
    echo 'hier het woord';

  2. Als je wilt dat php een code niet weergeeft in de browser kan je ervoor kiezen om er 2 // voor te zetten. Als je dit doet word de code niet uitgevoerd in de browser. Je kan ook meerdere codes tegelijk uitzetten door eerst de gewenste codes te selecteren en daarna Ctrl en / tegelijk in te drukken.

  3. Als je een woord wilt opschrijven met behulp van een variable gebruik dan de volgende code:
    $Hier de variable: kan wilekeurig woord zijn = 'Hier het woord dat je wilt schrijven';
    echo 'Hier de variable: hetzelfde als hierboven'


    eventueel kan je ook voor de variable een woord zetten zoals bijvoorbeeld:
    $name = 'Lars'
    echo 'Hello', $name


    uitkomst: Hello Lars


Php en html

  1. Om php in html te gebruiken moet je een openings- en sluitingstag gebruiken zoals in onderstaand voorbeeld:
    Openingstag:
    <?php

    Sluitingstag:
    ?>


    Naast openings- en sluitingstags te gebruiken hebben we ook nog een afkorting voor php echo. Namelijk:

    Lange versie:
    <?php echo

    Korte versie:
    <?=

    De = bij de korte versie staat dus voor php echo.

  2. Je kan met php ook in de URL werken. Dit kan je doen met de volgende code:
    $variable $_GET['name'];

    echo $variable;

    Voordat je iets in de url kan schrijven moet je ook nog wat tekst achter de documentnaam zetten namelijk:
    /?name=

    Je kunt hierbij in de url schrijven alsof het een html document is. Dat betekent dus dat je in de URL van de pagina (bijna) alle codes kan gebruiken.

    Je kan dit op twee manieren schrijven waarvan dit de tweede manier is:
    echo "Hoi", $_GET['Lars'];


PHP-logica van presentatie

  1. Met php kan je net als in html een ander document toevoegen aan jouw document. Gerbuik hiervoor de volgende code:
    require "Naam van document"


Arrays

  1. Je kan met php ook arrays maken met de volgende code:
    $Naam van array = [
    'woord',
    'woord',
    'woord'
    ];

    Je kan de array zolang maken als je wilt. In de array moeten woorden tussen aanhalingstekens staan maar cijfers hoeven dat niet.

  2. Om arrays uit te voeren zijn er ook verschillende mogelijkheden. voor deze opdracht gebruiken we deze array:
    $names = [
    'Lars',
    'Koen',
    'Liesbeth'
    ];


    • Je kan het op een manier doen waarbij je ze gewoon achter elkaar zet:
      foreach ($names as $name) {
      echo $name;
      };


    • Je kan het ook doen op een manier dat ze netjes onder elkaar staan.
      Hier zijn twee manieren voor.

      De eerste manier:
      <?php foreach ($names as $name) {
      <li>$name;</li>
      } ?>


      De tweede manier:
      <?php foreach ($names as $name) : ?>
      <li><?=$name;?></li>
      <?php endforeach; ?>

      mijn voorkeur gaat uit naar de eerste manier.


  3. Je kan met deze code iets toevoegen aan de array:
    $person[] = 'Lars';


Associative arrays

  1. een associative array is bijna hetzelfde als een gewone array maar aan deze arrays zit een sleutel aan, eigenlijk is het gewoon een beschrijving van wat je erachter zet.
    voor deze opdracht gebruiken we deze array:
    $names = [
    'Naam' => 'Lars',
    'Leeftijd' => 18
    ];


    bij dit voorbeeld zijn de Naam en de Leeftijd de sleutel, en Lars en 18 zijn het antwoord.


  2. Dit is een manier om het te laten zien in de browser zonder sleutel:
    <?php foreach ($person as $feature) : ?>
    <?= $feature; ?>
    <?php endforeach; ?>


    en dit is een manier om het te laten zien met sleutel:
    <?php foreach ($person as $feature => $value) : ?>
    <li><?= $feature; ?> <?= $value; ?></li>
    <?php endforeach; ?>


  3. Je kan met deze code iets toevoegen aan een array:
    $person['name'] = 'Lars';


  4. Die, var_dump en pre

    • Je kan een hele array onder elkaar zetten met pre:
      echo '<pre>';
      echo 'variable';
      echo '<pre>';


    • Je kan informatie laten verschijnen over de array met de code var_dump:
      var_dump(variable);

    • Je kan hier ook die bij gebruiken. die Zorgt ervoor dat de codes die onder deze code staan niet worden uitgevoerd.
      die(var_dump(variable));

    • Je kan die ook alleen gebruiken met echo ervoor:
      echo die;


      var_dump Kan je niet alleen gebruiken omdat ie dan iets moet hebben wat hij dan kan var dumpen.


Booleans

    Een boolean gebruik je vooral als wilt aangeven of iets goed of fout is. een boolean kent namelijk alleen maar true en false. Voor deze opdrachten gebruiken we deze array, en dan vooral de laatste namelijk: 'voltooid' => ''.
    $task = [
    'Titel' => 'Huiswerk afmaken',
    'Waneer' => 'Vandaag',
    'Wie gaat het doen' => 'Lars',
    'Voltooid' => ''
    ];


  1. Je kan dus een boolean beginnen door true of false te typen achter voltooid zoals onderstaand voorbeeld.
    $task = [
    'voltooid' => true
    of
    $task = [
    'voltooid' => false

    Maar zoals je dan kan zien in de uitslag komt er dan niks te staan.


  2. De volgende stap is om de array zichtbaar te maken met de code uit de vorige paragraaf:
    <?php foreach ($task as $feature => $value) : ?>
    <li><?= $feature; ?> <?= $value; ?></li>
    <?php endforeach; ?>

    Maar met een boolean is het eigenlijk makkelijker om deze manier te gebruiken:
    Titel: <? $task['Titel']; ?>


    Dit kan je dan bij elke feature doen, maar voor de boolean moet er ook nog iets extra bij.


  3. Bij de boolean beginnen we ook met de vorige code:
    Voltooid <? $task['Voltooid']; ?>

    Maar er moest nog iets bij voor de betekenis van de boolean.
    Voltooid <? $task['Voltooid']; 'completed' : 'incompleted'; ?>

    Als je nu true zet achter voltooid, komt er completed te staan en als je false neerzet komt er incompleted te staan. Je kan completed en incompleted gewoon veranderen in bovenstaande code.


Conditionals

    Een conditional heeft ook met de boolean te maken alleen is de conditional iets langer maar wel makkelijker te begrijpen.Je kan een conditional samen met een boolean gebruiken. We gaan ook weer dezelfde array gebruiken als bij de booleans.
    $task = [
    'Titel' => 'Huiswerk afmaken',
    'Waneer' => 'Vandaag',
    'Wie gaat het doen' => 'Lars',
    'Voltooid' => ''
    ];

  1. Hierbij gaan we ook weer met true en false werken en wat er moet komen te staan als dat zo is. Hieronder zie je een manier om een conditional te schrijven:
    if ($task['Voltooid']) {
    echo 'klaar';
    } else {
    echo 'nog niet klaar';
    }

    Deze code kan je achter de naam zetten zoals Voltooid of Titel.
    Deze code is handiger dan de kortere versie van de vorige paragraaf als je heel veel code hebt om te typen.
    Je kan ook icoontjes toevoegen. een voorbeeld hiervan is de site van w3 schools.
    zoek naar: unicode html en je vind het wel.

  2. Er is ook nog een manier om dit op te schrijven.
    <?php if ($task['Voltooid']) : ?>
    <span class='icon'>'klaar';</span>
    <?php else :
    'niet klaar';
    <?php endif; ?>

  3. Als je alleen iets wilt laten zien als het klaar of niet klaar is kun je deze code gebruiken:
    if (! $task['Voltooid']) {
    echo 'niet klaar';
    }


    het uitroepteken betekent in dit geval (niet) letterlijk.


Functions

  1. Hier zie je een function die we gaan gebruiken:
    function ($een, $twee, $drie)

    Je kan alle variable in de function var_dumpen. Bijvoorbeeld:
    function ($een, $twee, $drie) {
    var_dump($een, $twee, $drie);
    }

  2. Je kan de function een naam geven. bijvoorbeeld:
    function dumper($een, $twee, $drie)

  3. Je kan onder de function de function uitvoeren door de naam van de function op te schrijven zoals in onderstaan voorbeeld. Je kan hierbij dezelfde variable gebruiken maar je kan ook andere woorden gebruiken.

    Dezelfde variable
    dumper('een', 'twee', 'drie')

    Andere variable:
    dumper('Hallo', 'grote', 'wereld')
